Debug and Test with DevTools and VS Code

Objective

To test the webpage on different screen sizes and debug layout problems using Chrome DevTools and VS Code.

Steps Performed

  1. Opened the webpage in Google Chrome.
  2. Right clicked on the webpage and selected Inspect.
  3. Used the Device Toolbar to test mobile and tablet screens.
  4. Checked styles like width, padding, margin, and font size.
  5. Used the Network tab to check whether the CSS file loaded properly.
  6. Edited HTML and CSS files in VS Code and refreshed the browser.

Result

The webpage was tested successfully on different screen sizes and the layout worked properly.
